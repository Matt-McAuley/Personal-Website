About

I’m a Computer Science major at Cornell University who enjoys working with others to tackle interesting and complex problems with technology. I have extensive experience with full-stack development using tools such as React and Typescript on the fronted and C# and Python on the backend. I’ve worked on many projects including an API for organizing financial data, embedded software for controlling robots, an information retrieval system for exercises, and a fitness companion mobile app. I always strive for designing clean, maintainable, and well documented code, working effectively with others, and staying up to date with the latest trends in software.
